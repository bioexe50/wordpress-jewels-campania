🚀 Installazione Rapida

  1. Scarica il file ZIP cliccando il pulsante sopra
  2. Vai su Plugin → Aggiungi nuovo nel tuo WordPress
  3. Click su Carica plugin
  4. Seleziona il file ZIP scaricato
  5. Click Installa ora
  6. Click Attiva
ℹ️ Aggiornamento da v1.2.9?
Nessuna migrazione necessaria! Disattiva la vecchia versione, elimina i file, installa la nuova. Il database rimane invariato.

📋 Requisiti Sistema

  • WordPress: 5.8 o superiore
  • PHP: 7.4 o superiore
  • MySQL: 5.7+ / MariaDB 10.2+
  • JavaScript: Attivo nel browser

✨ Funzionalità Principali

  • 🗺️ Mappa Interattiva - Leaflet.js + OpenStreetMap
  • 📊 Upload CSV - Importazione massiva aziende
  • 🎯 Geocodifica Automatica - Integrazione Nominatim
  • 🔍 Filtri Avanzati - Provincia, città, settore, prodotto
  • 🔎 Ricerca Full-Text - Cerca per nome/indirizzo
  • 📍 Marker Clustering - Raggruppamento automatico
  • 🎨 Icone Personalizzate - Per settore aziendale
  • 📱 Responsive - Mobile e tablet friendly

📖 Utilizzo

Dopo l'attivazione, usa lo shortcode nella pagina o post:

[campania_companies_map]
